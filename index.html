Role: Plugin Developer / Unreal Engine Programmer
Engine: Unreal Engine

Tools: C++, Unreal Engine Plugin Development

Unreal Engine Plugin C++ VR Eye Tracking Dynamic Foveated Rendering Pupil Dilation Tool Development

A custom Unreal Engine plugin designed to support dynamic foveated rendering, eye gaze data, and pupil dilation data in Unreal Engine using the Bigscreen Beyond 2e and a fork of the Babellonia software. The goal of the project is to expose eye-tracking-related data in a way that can be used by Unreal systems, gameplay code, debugging tools, and future VR interaction features.

  • Worked on plugin-side Unreal Engine integration for eye gaze and pupil dilation data.
  • Explored support for dynamic foveated rendering workflows in Unreal Engine.
  • Focused on exposing technical data in a usable way for future gameplay, VR interaction, and performance-focused systems.
  • Used a forked software workflow to connect external eye-tracking functionality with Unreal Engine systems.
  • Built the project as a reusable Unreal plugin rather than a one-off project-specific implementation.
